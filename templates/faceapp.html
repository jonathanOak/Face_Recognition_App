Face Recognition Application


Face Recognition: This app is driven by a machine learning model. When the user uploads an image the app process the following:

  • Converts the image into grayscale;
  • Crops the Face;
  • Converts it into eigen image;
  • The model is processed;
  • A prediction is generated.
The functionality is dumped in the flask app.
